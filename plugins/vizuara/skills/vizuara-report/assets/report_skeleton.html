1
Executive summary

The engagement with Ramco began on 22 June 2026. It runs several workstreams in parallel: Phase 1 builds the Brain, Chia delivers the journey execution chatbot, and HRP covers HR and Payroll. Read one workstream at a time and it can look like cost keeps being added. Read them together, as this report does, and the shape is clear: the work is bounded and it ends.

Two facts anchor the rest of this document. First, the engine is accurate. On a set of 2,313 real questions it returned the correct core answer 92.3 percent of the time, with three hallucinations. Second, the build is nearly closed. Milestone 1 is complete, Milestone 2 is about half built, and the five engine milestones finish in the first week of September. The launch stays targeted for December.

Appendix
A
Method and definitions

Accuracy was scored by comparing each answer to a ground truth set. An answer that covers the ground truth counts as correct even when it says more, since a fuller answer is not a hallucination. A hallucination is counted only when the answer invents a fact that the source does not support. Inline identifiers look like pocrmn_ser_crt.
