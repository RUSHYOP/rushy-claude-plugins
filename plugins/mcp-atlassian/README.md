# mcp-atlassian

MCP-server plugin (packaged from RUSHYOP's system config, 2026-07-18).
Disabled by default — enable with `/plugin` -> mcp-atlassian@rushy per user/machine.

**Env:** ATLASSIAN_SITE — your Atlassian site, e.g. https://yourorg.atlassian.net (required). OAuth happens interactively on first use.
